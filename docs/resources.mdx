## Further Reading

### Textbooks

- **Probabilistic Robotics** — Thrun, Burgard, Fox (2005). The foundational textbook for robot perception and SLAM.
- **Modern Robotics** — Lynch, Park (2017). Comprehensive coverage of kinematics, dynamics, and control. [Free online](http://hades.mech.northwestern.edu/index.php/Modern_Robotics)
- **Robotics: Modelling, Planning and Control** — Siciliano et al. (2009). Advanced control theory for robotic arms.
- **Reinforcement Learning: An Introduction** — Sutton, Barto (2018). The standard RL textbook. [Free online](http://incompleteideas.net/book/the-book-2nd.html)
- **Deep Learning** — Goodfellow, Bengio, Courville (2016). Foundation for understanding neural networks. [Free online](https://www.deeplearningbook.org/)

### Key Research Papers

**Foundation Models for Robotics:**
- [RT-1: Robotics Transformer](https://arxiv.org/abs/2212.06817) — Google's first large-scale robot transformer (2022)
- [RT-2: Vision-Language-Action Models](https://arxiv.org/abs/2307.15818) — VLM directly outputs robot actions (2023)
- [PaLM-E: An Embodied Multimodal Language Model](https://arxiv.org/abs/2303.03378) — 562B embodied model (2023)
- [Octo: An Open-Source Generalist Robot Policy](https://arxiv.org/abs/2405.12213) — Open-source VLA (2024)
- [OpenVLA: An Open-Source Vision-Language-Action Model](https://arxiv.org/abs/2406.09246) — 7B VLA (2024)

**Simulation and Sim-to-Real:**
- [Isaac Gym: High Performance GPU-Based Physics Simulation](https://arxiv.org/abs/2108.10470) — NVIDIA's parallel RL framework (2021)
- [Sim-to-Real Transfer in Deep Reinforcement Learning](https://arxiv.org/abs/2009.13303) — Survey of transfer methods (2020)
- [Domain Randomization for Sim-to-Real Transfer](https://arxiv.org/abs/1703.06907) — Foundational domain randomization paper (2017)

**Humanoid Robotics:**
- [Learning Agile and Dynamic Motor Skills for Legged Robots](https://arxiv.org/abs/1901.08652) — ANYmal locomotion (2019)
- [Learning Humanoid Locomotion with Transformers](https://arxiv.org/abs/2303.03381) — Transformer-based humanoid control (2023)
- [DexMV: Imitation Learning for Dexterous Manipulation](https://arxiv.org/abs/2108.05877) — Multi-finger manipulation (2021)

### Online Courses

- [Self-Driving Cars Specialization](https://www.coursera.org/specializations/self-driving-cars) — University of Toronto on Coursera (perception, planning, control)

### Quick Setup Commands

```bash
# === ROS 2 Humble (Ubuntu 22.04) ===
sudo apt install ros-humble-desktop python3-colcon-common-extensions
source /opt/ros/humble/setup.bash
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc

# === Gazebo Harmonic + ROS 2 Bridge ===
sudo apt install gz-harmonic ros-humble-ros-gz

# === Python Virtual Environment for AI ===
python3 -m venv ~/robot_ai_env
source ~/robot_ai_env/bin/activate
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
pip install transformers accelerate octo-model

# === Isaac Sim via Docker ===
docker pull nvcr.io/nvidia/isaac-sim:4.2.0
docker run --gpus all -e "ACCEPT_EULA=Y" --network=host \
  nvcr.io/nvidia/isaac-sim:4.2.0

# === Workspace Setup ===
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws
colcon build
source install/setup.bash

# === Verify Everything ===
ros2 doctor                  # Check ROS 2 health
gz sim --version             # Check Gazebo version
python -c "import torch; print(torch.cuda.is_available())"  # Check GPU
nvidia-smi                   # Check GPU driver
```

### Common Environment Variables

```bash
# ROS 2
export ROS_DOMAIN_ID=0              # Isolate ROS 2 networks
export RMW_IMPLEMENTATION=rmw_fastrtps_cpp  # DDS implementation

# Gazebo
export GZ_SIM_RESOURCE_PATH=~/ros2_ws/src  # Model search path

# NVIDIA
export CUDA_VISIBLE_DEVICES=0       # Select GPU
export NVIDIA_VISIBLE_DEVICES=all   # Docker GPU access

# Python
export PYTHONPATH=$PYTHONPATH:~/ros2_ws/install/lib/python3.10/dist-packages
```
